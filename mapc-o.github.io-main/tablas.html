Cons los tags <table>, <tr>, <th> y <td> se crean tablas que organizan informaciòn para resumirla.

  • <tr>: Permite declara renglones (rows)
  • <th>: Declarar cabeceras de columna
  • <td>: Para declarar los datos de las celdas

Los atributos: border, colspan y rowspan

  • BORDER: Establece el ancho de borde para la tabla
  • CAPTION: Establece un titulo arriba d ela cabecera
  • COLSPAN: Permite que una celda ocupe varias columnas
  • ROWSPAN: Permite que una celda ocupe varios renglones
