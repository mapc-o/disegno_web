Listas

Funcionamiento

Las listas son usadas para resumir informaciòn y hacerla màs clara.
Las listas mejoran el SEO.
Existen del tipo ordenadas, no ordenadas y descriptivas

Listas ordenadas

Emplean la Etiqueta <ol> y la subetiqueta <li> que significa List Item

Ejemplo: Una lista de pasos a realizar:

  1. Sacar la llanta de refacciòn
  2. Sacar la llave de cruz
  3. Aflojar los birlos
  4. Quitar llanta ponchada
  5. Poner llanta refacciòn

Listas no ordenadas

Emplean la Etiqueta <ul> y la subetiqueta <li> que significa List Item

Ejemplo: Una lista de cosa sin orden especìfico:

  • Peras
  • Mangos
  • Plàtano
  • Manzana
  • Piña
