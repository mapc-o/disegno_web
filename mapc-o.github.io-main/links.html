Links e imàgenes

a

<a> Es un tag para incrustar imàgenes en la pàgina. Sus atributos principales son:

  • title
  • href
  • target

Opciones de target:
  • _SELF: Abre el link en la misma pàgina
  • _BLANK: Abre el link en una nueva pàgina del navegador
  • _PARENT: Abre en la pàgina contenedora de la actual
  • _TOP: La abre en la parte superior de la pàgina parent
Ejemplo: Sitio de Github

Imàgenes

El tag <img> sirve para incrustar imàgenes en la pàgina.

Sus atributos importantes son:

  • Title: Es el tooltip
  • Src: Es la URL a la imagen, ya sea relativa o absoluta
  • Width: Ancho en pìxeles
  • Height: Alto en pìxeles
  • Loading: Forma de carga: Lazy o eager
